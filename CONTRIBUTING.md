# Contributing — Help Beginners Learn Python

Thank you for improving this **beginner-friendly** Python learning repository!

Every contribution you make helps someone who is learning Python for the first time.

## How to Contribute

1. **Fork** this repository on GitHub
2. **Clone** your fork to your computer
3. Create a branch: `git checkout -b beginner-string-examples`
4. Make your changes
5. **Commit** with a clear message: `Simplify string examples for beginners`
6. **Push** and open a **Pull Request**

## What We Need Most

- **Simpler explanations** — plain English, no jargon
- **More beginner exercises** with small, runnable examples
- **Comments in code** that explain what each line does
- **Fixed versions** as `*_fixed.py` when correcting practice bugs
- **Typo fixes** in README and lesson guides
- **New beginner modules** — lists, dicts, loops, functions, file reading

## Writing for Beginners

When adding code or docs, ask:

- Would someone with **zero coding experience** understand this?
- Is the example **short** (under 30 lines when possible)?
- Did you use **clear names** like `student_name` instead of `x`?
- Does the file **run without errors**?

Follow [PEP 8](https://peps.python.org/pep-0008/) style, but keep readability first.

## Pull Request Checklist

- [ ] Code runs: `python your_script.py`
- [ ] Language is beginner-friendly (simple words)
- [ ] README or module guide updated if you add new files
- [ ] No secrets, passwords, or personal data in commits

## Questions?

Open an [issue](https://github.com/NadirBakhsh/full-stack-ai-with-python/issues) — beginners and contributors are welcome!
